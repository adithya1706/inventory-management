# Welding Inventory Management System

## 📌 Project Description

This project is a web-based Inventory Management System developed to manage welding materials and stock efficiently. It allows users to add, update, search, and monitor inventory items.

## 🚀 Technologies Used

* HTML
* CSS
* JavaScript
* Firebase (Authentication & Firestore Database)

## ✨ Features

* User Login Authentication
* Add new inventory items
* Update existing items
* View stock details
* Search functionality
* Low stock indicator

## ▶️ How to Run the Project

1. Open the project folder
2. Run the `index.html` file in a browser
3. Login using Firebase authentication
